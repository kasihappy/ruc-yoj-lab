# 题目8：计算学分绩点

## 题目描述
给定期末总评成绩，计算所得的学分绩点。

## 绩点对照表
| 百分制 | 等级制 | 绩点 |
|--------|--------|------|
| 90-100 | A | 4.0 |
| 86-89 | A- | 3.7 |
| 82-85 | B+ | 3.3 |
| 78-81 | B | 3.0 |
| 75-77 | B- | 2.7 |
| 72-74 | C+ | 2.3 |
| 68-71 | C | 2.0 |
| 64-67 | C- | 1.5 |
| 60-63 | D | 1.0 |
| 0-59 | F | 0 |

## 输入格式
一个0~100的整数，代表期末总评成绩

## 输出格式
该成绩对应的学分绩点

## 数据限制
- 内存限制：256MiB
- 时间限制：1000 ms
